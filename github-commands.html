Github basics

  1. Add Files

    git add file name

    git add . (adds all files in the current dir)

    git add *.* (adds all files in the current dir)

    git add *.html (adds only .html files)

  2. Commit Files

    git commit -m"Some message"

    Ex: git commit -m"Initial commit"

  3. Push Files

    git push --set-upstream origin master (only for the first time)

    or

    git push -u origin master (only for the first time)

    git push (after the first commit and push)
